Q1. Which protocol does traceroute use by default on most Linux systems?
 a) ICMP Echo Request
 b) TCP SYN
 c) UDP datagrams
 d) HTTP GET
Answer: c)) UDP datagrams

Q2. Which traceroute option displays numeric IPs only, without hostnames?

 b) -h
 c) -N

Answer: a)) -n

Q3. What flag forces traceroute to use ICMP Echo Request packets?

Answer: a)) -I
